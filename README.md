# AOSP setup
Quick start my aosp sync up
```
git clone https://github.com/stuartore/aosp-setup.git
cd aosp-setup && bash aosp.sh
```

### Custom
```
bash aosp.sh ${ROM_manifest_url}

# example
bash aosp.sh https://github.com/xxxxx/manifest.git
```
